URL Shortener Microservice

How to use:

  • Copy/paste or type the absolute path of the URL into the input box below.
  • You will receive a JSON object of form: { original_url : [original_url], short_url : [short_url]
    This can be used as an API endpoint. Alternatively, you can just copy [short_url].
  • To visit your original URL with the shortened URL, append [short_url] to the subdirectory of the API endpoint
    eg. https://glossy-clownfish.glitch.me/api/shorturl/[short_url]
